# ThymeCMS

ThymeCMS is the custom made Content Management System for the Thyme Recipe Blog. 
It's a lightweight system that allows the the management of recipes.
Nutrition Data is sourced from USDA and utilizes OpenAI to process Nutrition information (experimental)

USDA: https://fdc.nal.usda.gov/
OpenAI: https://openai.com/

This project relies on the ThymeBackend to run. 
Instructions for setup assume the backend is running.

Running the application:
  npm run dev
